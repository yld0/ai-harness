# Workspace skills

Drop skill folders here, each of the form:

```text
<name>/SKILL.md
```

`SKILL.md` must include YAML front-matter with at least `name` and `description`. Skills in this tree override identically named skills from `references-skills/` and `~/.ai/skills/`.

 See `DESIGN.md` and `docs/openclaw-skill-discovery.md` in the repository for the full contract.
